# button

GUI app for remembering hotkeys

## Why Not Raycast Extension?

Technically, Button could work as a simple Raycast extension, with the possibility of using it in Vicinae on Linux aswell.

I wanted to learn how to make a lightweight, non-Electron GUI app using Go/Svelte, and create pipelines for releasing and publishing it.\
By creating Button as a Raycast extension, even though the development would be far simpler,\
I'm both limiting the user base to Raycast users only, and limiting the potential functionality of the app.
